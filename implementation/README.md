# 구현

## 백준 1000번 A+B

[Problem](https://www.acmicpc.net/problem/1000) / [Solution](boj1000.cpp)

### 문제 요약

두 정수 A와 B를 입력받고, A+B를 출력합니다.

### 입력 조건

- A와 B를 공백 한 칸으로 구분하여 입력받는다.
- A와 B는 0보다 크고 10보다 작은 자연수이다.
